crawlability
============

[![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

Check crawlability of web sites.


Development
-----------

### Requirements

* Python 3.4+

### Setup

```sh
git clone https://github.com/capybala/crawlability.git
cd crawlability
pyvenv venv
. venv/bin/activate
pip install -r requirements.txt
```

### Run server

```
python app.py
```
